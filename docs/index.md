# Skillable演習トラブル (マイクロソフト公式カリキュラム)
- Entra IDの情報は、演習ガイドのまん中タブ[Resources]にある
- 演習ガイドのページ全体を翻訳機能で日本語にするとコピー&貼り付けがうまくいかない(選択部分だけを翻訳することは問題ない)
- Entra IDテナントと仮想マシンなどのリソースは紐付くので、テナントを追加すると切り替えが必要
- キーボードが英語(英→日対応)

	| 英語 | 日本語 |
	|-----|-----|
	|@| シフト2|
	|\\ | ] (む) (Enterの左の下の方)|
	|'| : (コロン)|
	|"| シフト : (コロン)|
	|:| シフト ; (セミコロン)| 
	|=| ^ | 

- キーボード入力ができない

	演習を閉じて再接続

	それでもだめなら演習環境の再起動
  
- 演習環境の再起動

	演習ウィンドウ左上の[💻(コンピューターマーク)]-[Reset/Reboot]-[Yes]
  
- 文字が2つ入る

	[半角/全角]を1回押す (駄目なら何度か繰り返す)
  
	[半角/全角]を押して日本語モードにしたあと、通知エリアの[あ]をクリックして[A]の英数字モードにすると直る場合もある

- 解像度が下がってしまった場合

	演習環境のディスプレイのプロパティで1024x768以上に設定

- クラウドシェル内のクリップボード操作

	|操作|キー|
	|----|----|
	|コピー|Ctrl-Ins|
	|貼り付け|Shift-Ins|
	|切り取り|Shift-Del|

- 演習環境のホストから、演習マシンに文字列を貼り付ける手順

	1. 演習仮想マシン内部でメモ帳などを起動
	1. 演習ウィンドウ左上⌨(キーボードマーク)をクリック
	1. [Type Text]-[Type Clipboard Text]を選択
	1. 演習仮想マシンのクリップボード経由で文字列が貼り付く (あまり文字数が多いと取りこぼしがある)

※演習環境の設定によっては、表示されたダイアログボックスに文字列を貼り付けてから転送する場合もある
